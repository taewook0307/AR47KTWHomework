1. 키워드
  c++에서 키워드는 컴파일러가 지정한 예약어
  궁극적으로 본다면 그런 예약어 전체를 키워드라고 부릅니다.
  우리는 키워드를 변수명이런것으로 사용할 수 없음
  예약되어 있고 이걸 사용하면 컴파일러는 변수가 아니라 예약어로 인식
  https://learn.microsoft.com/ko-kr/cpp/cpp/keywords-cpp?view=msvc-170
  int int; 컴파일러가 ????????? <= 에러가 나게 됩니다.

2. 자료형 == 타입
  정수 실수 문자 이런것들을 구분하기 위한 키워드로 자료형이라는 하위분류
  int <= 정수

3. 주소값 == 위치 == 이름
  실행됐다면 모든 건 램에서 존재
  모든 이름은 '램에서의 n바이트번째'를 의미

4. 인터페이스
  내가 짠 코드가 아닌 다른 프로그래머가 짠 코드를 쓰는데 그 프로그래머가 우리에게 사용방법을 제시하는 모든 것
  = 쓰는 방법(클래스, 함수, 변수 등등)

5. std
  스탠다드 라이브러리를 의미
  c++ 기본으로 유저에게 제공되는 코드들과 클래스 인터페이스들의 집합

6. std::vector
  스탠다드 라이브러리에서 제공하는 동적배열을 위한 인터페이스(클래스)

7. stl
  스탠다드 탬플릿 라이브러리의 약자
  std에 속한 기능들 중에서 템플릿을 사용하는 기능들
  자료구조에서만 템플릿을 사용하는게 X => stl == 자료구조 : 잘못된 말
